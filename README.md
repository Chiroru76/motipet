#  🎮 サービス概要

### 〜日々の習慣やタスクの達成が、ペットの成長に直結する育成ゲーム型のタスク管理アプリ〜

勉強・運動など習慣や日々のTODOを達成すると経験値を獲得し、ペットがレベルアップ / 進化する一方で、
継続的にタスクを達成しお世話を続けないと、ペットが弱り「死」を迎えてしまいます。
タスクを達成するとペットが褒めてくれたり、LINE通知によるリマインド機能などにより、
育成ゲーム感覚で習慣形成やタスクの管理を行うことができるサービスです。


#  🌱サービス開発の背景

私はランニングや勉強など日々の習慣として行なっていることのログを記録するようにしています。
それは、記録として残すことで、数値として自分の行動や積み重ねてきた努力が「可視化」されることで、
達成感や自己効力感を感じ、それが習慣を継続するモチベーションの一つになっているからです。

また、このような日々の努力の積み重ねや行動は、ゲームで主人公が経験値をゲットし成長していく
様子と重なる感覚があるため、楽しみながら続けることができていると感じています。

こうした経験から日々の行動がの結果が直接ペットの成長・進化という形で可視化できるゲーミフィケーション型のアプリを作りたいと考えました。


# 👤ユーザー層について

- **「たまごっち」をはじめとする育成ゲームに馴染みのある方**
  ペットの成長やコレクション要素に懐かしさや楽しさを感じてもらいやすい。
  ゲーム感覚で習慣形成やタスク管理を取り入れることができる

- **今時点で何か毎日の習慣として続けていることがある、または新たに身につけたい習慣がある方**
  ペットの育成を、習慣を続ける理由の一つとして加えることで、違う面でのモチベーションが生まれ
  行動が継続しやすくなる。

- **味気ないタスク管理に物足りなさを感じる人**
  すでにメモアプリや TODO アプリを使っているに対し、完了したタスクを削除する又はチェックをする
  達成感に加えて、タスクの完了をペットのレベルアップ、進化に結びつけることで達成感が強化され、
  単なるタスク管理が楽しい体験に変わる。


# 📝サービスの利用イメージ

### 🎯【利用イメージ】

- 新規登録時に「たまご」を入手。初めてのレベルアップ時にペットが誕生する。
- ユーザーは日々の習慣（例：勉強、運動、読書等）や TODO を登録する。
- タスクを完了すると、ペットが経験値を獲得してレベルアップ、一定条件を満たすと進化する。
- 一方で、一定期間ペットの育成を怠るペットが死亡してしまうが、新しい「たまご」から育成を再スタートできる。

### 🌟【得られる価値】

- **ゲーム感覚で自己管理**
  　自らの習慣の見直しや継続、タスクの管理を「ペット育成」という形に置き換えることで、
  　自然に楽しみながら行うことができる。

- **フィードバックによるモチベーションの向上**
  　自分の行動がペットの成長・進化あるいは死に直結するため、行動のモチベーションが生まれる。
  　また、習慣とTODOの達成数や行動量がグラフやカレンダーを通して可視化されることで、
  　達成感や自己肯定感を高めることができる。

### 🥚「たまご」の入手
[![Image from Gyazo](https://i.gyazo.com/bb70c181a8495e9928aada1c3f172f73.gif)](https://gyazo.com/bb70c181a8495e9928aada1c3f172f73)
  - 新規登録時に「たまご」を入手
  - 初回レベルアップ（Lv.1→Lv.2）の際に「たまご」からペットが誕生


### 📈 成長・進化
  - タスク完了時に獲得する経験値によりレベルアップ
  　※ 獲得できる経験値はタスク登録時に設定した「難易度」に応じる
  - Lv.10のレベルアップ時に進化
  - ペットの誕生・進化時に表示されるモーダルからXへシェアが可能


### 💀 ペットの死亡
  - 毎日自動で「しあわせ度」が減少（定期バッチ処理）
  - しあわせ度が0になるとペットが死亡するため、定期的に「えさ」をあげる必要がある
  - ペットが死んでしまった場合は「たまご」の状態から育成を再スタート
